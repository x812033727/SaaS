設定教學

  1. 登入 MiniMax Developer Platform，建立 API Key；Token Plan 使用方案頁提供的 Token Plan Key。
  2. 模型建議使用 MiniMax-M2.7；需要較低延遲可用 MiniMax-M2.7-highspeed。
  3. 儲存後按「測試 MiniMax 連線」。測試會啟動單輪 Claude Agent SDK session，可能產生極少量用量。
  4. 到店家「AI 客服」頁及 LINE 分別測試，回答來源顯示 Claude Agent SDK 即代表已啟用。

移除設定後會安全退回 FAQ 規則模式，既有 FAQ 不會遺失。
