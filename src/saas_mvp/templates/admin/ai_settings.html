設定教學

  1. 登入 Anthropic Console，先完成帳務與用量上限設定。
  2. 到 API Keys 建立平台專用 Key；不要與個人開發環境共用。
  3. 從 Anthropic 模型文件複製要使用的模型 ID。
  4. 在本頁貼上 API Key 與模型 ID，儲存後按「測試 Anthropic 連線」。測試會啟動一個單輪 Claude Agent SDK session，可能產生極少量 API 費用。
  5. 到店家「AI 客服」頁輸入測試問題，再從 LINE 實際測試；回答來源顯示 Anthropic 即代表正式 AI 已啟用。

建議在 Anthropic Console 設定每月預算與警示。移除設定後會安全退回 FAQ 規則模式，既有 FAQ 不會遺失。
