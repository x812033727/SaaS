# 產品藍圖：saas-Line bot

## 願景

打造可持續擴充的 SaaS LINE Bot 平台，讓商家快速上線並持續迭代自動化對話服務

## 目標用戶

中小商家與營運人員，透過 LINE 官方帳號做客服自動回覆、推播與簡單交易

## 核心功能

- **[P0]** Webhook 接收 — 穩定接收 LINE 訊息事件並驗證簽章
- **[P0]** 自動回覆引擎 — 依關鍵字／規則回覆文字與圖文選單
- **[P0]** 後台管理介面 — 商家可設定回覆規則與查看對話紀錄
- **[P1]** 推播訊息 — 對指定客群群發或排程推播
- **[P1]** 用戶分群與標籤 — 依互動行為標記名單供精準推播
- **[P1]** 數據儀表板 — 訊息量、回覆率、活躍用戶等基本指標
- **[P2]** 多帳號管理 — 單一後台管理多個 LINE 官方帳號
- **[P2]** AI 智能回覆 — 接 LLM 處理非規則型自然語言詢問

## 里程碑

- M1 Webhook 接收 + 自動回覆引擎 + 後台管理介面（可上線基本客服自動回覆）
- M2 推播訊息 + 用戶分群標籤 + 數據儀表板（具備行銷觸達與成效追蹤）
