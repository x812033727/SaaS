# 產品藍圖：saas-line-translate-bot

## 願景

讓任何 LINE 群組即時跨語言溝通，企業可在自有 SaaS 下一鍵啟用多租戶翻譯機器人

## 目標用戶

跨國團隊、客服與電商賣家，在 LINE 對話中需要即時雙向翻譯而不切換工具

## 核心功能

- **[P0]** LINE Webhook 接收 — 驗證簽章並接收 Messaging API 訊息事件
- **[P0]** 訊息翻譯 — 呼叫翻譯引擎將收到文字譯為該租戶設定的目標語言並回覆
- **[P0]** 多租戶管理 — 每租戶獨立設定自己的 LINE channel（secret/token）與目標語言
- **[P0]** 租戶路由 — 依 destination/channel 正確對應到租戶設定與憑證
- **[P1]** Quota 與用量計量 — 每租戶設翻譯字數/則數上限，超額擋下並記錄用量
- **[P1]** 來源語言自動偵測 — 自動判斷輸入語言，避免同語言重複翻譯
- **[P1]** 管理 API/介面 — 租戶可自助新增、查詢、更新 channel 與語言設定
- **[P2]** 多目標語言 — 單一群組同時輸出多種語言譯文
- **[P2]** 翻譯快取 — 相同文字快取結果以降低 API 成本與延遲
- **[P2]** 用量儀表板 — 視覺化各租戶用量與 quota 餘額

## 里程碑

- M1 單租戶可跑通：Webhook 接收＋簽章驗證＋訊息翻譯回覆＋基本租戶設定（channel 與目標語言）
- M2 多租戶化：租戶路由、自助管理 API、Quota 計量與來源語言自動偵測
